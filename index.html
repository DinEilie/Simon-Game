Simon Game

Information

Simon is a game of short-term memory skill (originally invented by Ralph H. Baer and Howard J. Morrison).

How to play:


While playing series of tones and lights (Red ,Green ,blue and Yellow), the user is required to repeat the sequence. If the user succeeds, the series becomes progressively longer and more complex. Once the user fails the game is over.

Difficulties:


Easy - every new level the game pattern sequence will play again with flashy lights.

Medium - every new level the game pattern sequence will play again faster with dimmed lights.

Hard - every new level only the new color that has been added to the sequence will play and with dimmed lights.
